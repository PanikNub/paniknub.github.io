PanikHub

Description: PanikHub is an open-source terminal program written in C that offers extremely lightweight games that can run on any device (if you have what it takes). Note that running PanikHub will require some knowledge in programming and that PanikHub is still in development! So expect bugs!
  • Download
  • Github Repository
How to run PanikHub: Download a C compiler (preferably gcc) and if you don't know how to, I would recommend taking a moment to learn C. Once you got C, extract Panikhub.zip, compile main.c and execute a.out/a.exe (depending on your operating system).
